# Описание проекта для резюме

**Название:** Система поддержки выбора поставщика

**Кратко:** Flask-приложение для многокритериального выбора связки «завод + транспортная компания» с учетом стоимости доставки, времени, надежности, расстояния и логистических ограничений.

**Что реализовано:**

- веб-интерфейс для ручного ввода данных о заказчике, заводах, перевозчиках и ограничениях;
- импорт данных из CSV/XLSX/XLS;
- автоматическое формирование альтернатив «завод + транспортная компания»;
- расчет объема, веса, числа фур, стоимости, времени доставки и совокупной надежности;
- фильтрация альтернатив по tracking, тарифу, времени и надежности;
- расчет весов критериев тремя методами: дисперсионный, энтропийный, AHP;
- ранжирование альтернатив методом TOPSIS;
- формирование итогового плана поставки по каждому продукту;
- опциональная визуализация маршрутов на карте через Yandex Maps API.

**Стек:** Python, Flask, Pandas, HTML, CSS, JavaScript, Bootstrap.

**Формулировка для резюме:**

> Разработал Flask-приложение для поддержки выбора поставщика в логистической задаче. Реализовал импорт данных из таблиц, расчет логистических показателей, фильтрацию альтернатив, расчет весов критериев методами AHP, энтропии и дисперсии, а также ранжирование поставщиков методом TOPSIS. Приложение формирует итоговый план поставки по каждому продукту и поддерживает опциональную визуализацию маршрутов на карте.
